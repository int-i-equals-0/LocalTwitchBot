## О проекте

**[LocalTwitchBot]** — это локальный Twitch-бот для стримеров, который предоставляет привычный набор функций облачных ботов, но работает полностью на вашем компьютере.

Современные стримеры часто зависят от облачных сервисов вроде StreamElements, NightBot или WizeBot. Такие решения удобны, но имеют недостатки: возможные сбои, задержки, ограничения кастомизации и зависимость от сторонней инфраструктуры.

Этот проект решает данную проблему: бот запускается локально, хранит конфигурацию на вашем устройстве и не требует доверия к внешним облачным сервисам.

Основная цель проекта — дать стримерам **полный контроль**, **стабильность**, **приватность** и **простоту настройки** без необходимости писать код.

---

## Основные возможности

- **Команды чата**  
  Создание пользовательских команд для взаимодействия со зрителями.

- **Награды за баллы канала**  
  Автоматическая обработка наград Twitch Channel Points.

- **Оповещения о событиях**  
  Реакция на подписки, фолловы, рейды и другие события канала.

- **Shoutout**  
  Быстрое упоминание других стримеров в чате.

- **Периодические действия**  
  Автоматическая отправка сообщений или выполнение действий по расписанию.

- **Фильтрация запрещённых слов**  
  Модерация чата с помощью настраиваемого списка банвордов.

- **Локальная работа**  
  Бот работает на вашем компьютере и не зависит от сторонних облачных сервисов.

- **Удобный веб-интерфейс**  
  Самодокументируемый и интуитивно понятный UI позволяет настраивать бота без технических знаний.

- **Приватность и контроль**  
  Конфигурация и данные хранятся локально у стримера.

---

## Почему локальный бот?

| Облачные боты | Локальный бот |
|---|---|
| Работают на сторонних серверах | Работает на вашем компьютере |
| Возможны сбои и задержки | Меньше зависимости от внешних сервисов |
| Ограниченный контроль над инфраструктурой | Полный контроль над запуском и данными |
| Конфигурация хранится у провайдера | Конфигурация хранится локально |
| Требуют доверия к стороннему сервису | Вы сами контролируете окружение |

## Начало работы

> Ниже приведён примерный шаблон инструкций. Замените команды и пути в соответствии с вашей реализацией.

### Требования

- Аккаунт [Twitch](https://www.twitch.tv/)
- Установленный [Node.js](https://nodejs.org/)
- Права модератора или доступ бота к чату канала
- Токены доступа Twitch API / OAuth

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/int-i-equals-0/LocalTwitchBot.git

# Перейдите в папку проекта
cd LocalTwitchBot

# Установите зависимости
install.bat или install.sh, в зависимости от вашей ОС
```

### Запуск

```bash
сначала
start-server.bat

затем
start-client.bat
```

После запуска откройте веб-интерфейс в браузере, например:

```text
http://localhost:3000
```

Если порт отличается, используйте тот, который указан в консоли после запуска.

## Веб-интерфейс

Веб-интерфейс разработан так, чтобы быть понятным даже для нетехнических пользователей.

С его помощью можно:

- создавать и редактировать команды чата;
- настраивать автоматические ответы;
- управлять фильтрами запрещённых слов;
- настраивать события подписок, фолловов и рейдов;
- создавать периодические сообщения;
- настраивать награды за баллы канала;
- просматривать текущее состояние бота.

Интерфейс содержит встроенные подсказки, поэтому большинство настроек можно выполнить без чтения дополнительной документации.

## Как это работает

Условно проект можно разделить на несколько частей:

1. **Ядро бота**  
   Подключается к Twitch и обрабатывает события чата.

2. **Модуль автоматизации**  
   Выполняет команды, периодические действия, реакции на события и модерацию.

3. **Веб-интерфейс**  
   Позволяет управлять конфигурацией бота через браузер.

4. **Локальное хранилище**  
   Сохраняет настройки, команды, фильтры и другие параметры на вашем устройстве.

## Безопасность

- Токены и конфиденциальные данные хранятся локально.
- Рекомендуется использовать отдельный аккаунт для бота.
- Не передавайте свои OAuth-токены третьим лицам.
- Не коммитьте `.env` и другие секретные данные в публичный репозиторий.
- Используйте только официальные механизмы Twitch API и соблюдайте правила Twitch.

---

## Ограничения

- Для работы бота необходим доступ к Twitch.
- Компьютер, на котором запущен бот, должен быть включён, пока бот должен быть активен.
- Функциональность может зависеть от прав бота на канале и выданных OAuth-разрешений.
